NIKO KADI is based on and inspired by the traditional game of poker, but incorporates unique elements of the Kenyan Culture and gameplay mechanics. The game is designed for 2-8 players and features a ton of rule variations. This game allows YOU, the player, to define your own poker rules, bet on wins and the best part of it all, walk away with cash prizes.

Features

In Game Communication

Talk to other players in the game continuously and in realtime

Supports upto 8 players

The game support up to a maximum of 8 players and a minimum of 2 players

Chat Support

The game allows in game communication via chat. User's can also send messages to each other via the inbox feature

M-PESA

Integration with DARAJA API allows users to deposit to their account and withdraw money they've won in a game

Spotify

Spotify allows players to play their own music during a game

Rule Variations

The game allows players the chance to define and amend the game rules. This allows a ton of variation

Bet and Win

The game allows users to place a stake on a game and if they win the game, they walk away with the total stake

Rules for Playing with Joker
  • Each Joker will make your opponent pick 5 cards only
  • Joker can be blocked by Ace
  • Both Jokers can cancel a special request by Ace
  • A Joker followed by Bomb will have no effect. Only the bomb will apply
